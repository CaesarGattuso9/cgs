Rust Performance Backend

最初的博客后端是用 Node.js + Express 写的，随着流量增长，P99 延迟开始飙升。决定用 Rust 重写，选择了 Actix-Web 框架。

// Cargo.toml [dependencies] actix-web = "4" sqlx = { version = "0.7", features = ["postgres", "runtime-tokio"] } serde = { version = "1", features = ["derive"] }

迁移后，同等负载下 CPU 使用率从 80% 降至 12%，内存占用从 380MB 降至 28MB，P99 延迟从 340ms 降至 31ms——提升超过 10 倍。
